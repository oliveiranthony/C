## ⚙️ Funcionamento do Programa

Para cada funcionário, o sistema:

1. Solicita o salário mensal
2. Registra a frequência diária
3. Conta o número de faltas
4. Conta o número de atestados
5. Calcula o desconto proporcional ao número de faltas

O valor do desconto é calculado dividindo o salário mensal pela quantidade de dias do mês, descontando esse valor a cada falta registrada.

---

## 📥 Entrada de Dados

O usuário informa:
- Quantidade de funcionários
- Quantidade de dias do mês
- Salário mensal de cada funcionário
- Frequência diária (`0` ou `1`) para cada funcionário

---

## 📤 Saída de Dados

O programa exibe, para cada funcionário:
- Total de faltas no mês
- Total de atestados entregues
- Valor total descontado do salário
