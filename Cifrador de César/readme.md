# 🔐 Cifrador de César em C

Este projeto implementa o clássico **Cifrador de César**, um método simples de criptografia onde cada letra de uma mensagem é deslocada um número fixo de posições no alfabeto.

O projeto foi desenvolvido em **linguagem C** com o objetivo de praticar manipulação de **strings, caracteres e lógica de programação**.

---

## 📌 Objetivo do projeto

Este projeto foi criado com o objetivo de praticar:

- Manipulação de **strings em C**
- Estruturas de repetição
- Manipulação de **caracteres e ASCII**
- Lógica de criptografia simples

---

## 🧠 Sobre o Cifrador de César

O Cifrador de César é uma técnica de criptografia usada na Roma Antiga por **Júlio César** para enviar mensagens secretas aos seus generais.

Ele consiste em substituir cada letra por outra que esteja um número fixo de posições à frente no alfabeto.

---

## 📚 Como funciona

O usuário digita uma palavra ou frase e um **valor de deslocamento**.  
O programa então desloca cada letra da mensagem no alfabeto.

### Exemplo

Entrada:

Mensagem:
casa

Deslocamento:
3

Saída:
fdvd
