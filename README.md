# 🔵 Linguagem C

Bem-vindo ao meu repositório de **Linguagem C** 👋

Aqui compartilho meus **estudos, exercícios e pequenos programas desenvolvidos em C**, focados no fortalecimento da **lógica de programação**, entendimento do funcionamento do código em baixo nível e construção de uma base sólida para outras linguagens.

---

## 📚 Sobre o Repositório

Este repositório reúne atividades desenvolvidas durante meus estudos em **Linguagem C**, incluindo exercícios acadêmicos e práticas voltadas para:

* Estruturas de controle (if/else, loops)
* Funções
* Vetores e matrizes
* Manipulação básica de dados
* Raciocínio lógico e resolução de problemas

A Linguagem C tem papel fundamental na minha formação, pois ajuda a compreender como os programas realmente funcionam “por baixo dos panos”.
